## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd lynon-solution

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Build for Production

```sh
# Build the project
npm run build

# Preview the production build
npm run preview
```
